## 🎮 Características
- 🧩 Tablero clásico de buscaminas
- 🚩 Sistema de banderas para marcar minas
- ⏱️ Contador de tiempo y minas restantes
- 🎚️ Diferentes niveles de dificultad
- 🔊 Efectos de sonido incluidos
